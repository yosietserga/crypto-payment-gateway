API Documentation

Comprehensive guide to integrate with our Crypto Payment Gateway API

Introduction

Our Crypto Payment Gateway API allows you to accept cryptocurrency payments and process payouts programmatically. This guide explains how to interact with our API endpoints.

Base URL

Production: https://eoscryptopago.com/api/v1

Sandbox: https://eoscryptopago.com/sandbox/api/v1

Authentication

All API requests must be authenticated using API keys. Include your API key in the Authorization header:

HTTP Header
Authorization: Bearer YOUR_API_KEY

Content Type

All requests should use JSON format:

HTTP Header
Content-Type: application/json
